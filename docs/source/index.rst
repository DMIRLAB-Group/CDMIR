Welcome to causal-dmir's documentation!
=======================================

**causal-dmir** is a python package. It offers the implementations of some classical and frontier algorithms related to causality.

.. note::
   This project is under active development. For source code, please kindly refer to our `GitHub Repository <https://github.com/DMIRLAB-Group/causal-dmir>`_.
